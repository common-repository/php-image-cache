== Description ==
This plugin cache images with PHP. Other cache plugins depends on your server settings. If you don't have the appropriate server settings enabled, then your images will not be cached with the other cache plugins.

With this plugin you are forcing the images to be cached and your page will load much faster for returning visitors.
 For more information see installation page or the 
[Webfish](http://www.webfish.se/wp/plugins/php-image-cache "Webfish - Wordpress plugin") homepage.

== Installation ==

This is some brief installation notes. For more information see [Webfish](http://www.webfish.se/wp/plugins "Wordpress plugins")

1. Upload the plugin source to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
